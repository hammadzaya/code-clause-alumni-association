🎓 Alumni Association Portal

A Full-Stack Alumni Association Web Portal developed during my internship at CodeClause.
The platform enables alumni and administrators to connect, share job opportunities, organize events, and participate in discussions through a centralized digital portal.
This project demonstrates end-to-end full-stack development using HTML, CSS, JavaScript, PHP, and MySQL with modular architecture and responsive UI design.

🚀 Features
🔐 User Authentication

Secure user registration and login
Session-based authentication
Input validation for security

👩‍🎓 Alumni Registration

Alumni can create profiles
Store professional details, graduation year, and contact information
Manage personal information

💼 Job Listings

Alumni can share job opportunities
Users can browse and view job postings

📅 Event Management

Admin can create, update, and manage alumni events
Users can view upcoming events

💬 Forum Discussions

Interactive discussion forum
Users can create threads and reply to posts
Encourages alumni engagement and knowledge sharing

🛠 Admin Dashboard

Manage users, jobs, events, and forum discussions
Centralized control panel for portal management
